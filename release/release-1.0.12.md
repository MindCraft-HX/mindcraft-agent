MindCraft-Agent v1.0.12 更新日志

新功能
- 会话级模型选择：每个会话可独立选择模型和推理强度，不再强制全局共享
- 会话自定义指令：支持为每个会话设置独立指令，Agent 将始终遵循
- 会话指令支持引用文件附件，Agent 可读取指定文件内容辅助理解

问题修复
- 修复会话指令被覆盖导致指令丢失的问题
- 修复会话指令设置后不生效的问题
- 修复 CodeX 文件变更预览和多次 diff 渲染不一致的问题
- 修复 CodeX 用户气泡代码块语法高亮配色异常的问题
- 修复 Markdown 渲染后外部链接无法打开的问题
- 修复 CodeHub 顶部 Tab 排序不稳定的问题
- 修复跨会话模型配置互相污染的问题
- 修复会话模型继承失效的问题
- 修复用户消息代码块高亮在亮色主题下配色不正确的问题
- 加固会话存储和配置安全性

功能改进
- 默认运行模式改为自动编辑，更符合日常使用习惯
- Agent 通知声音体验优化
