# MindCraft-Agent v1.0.11 更新日志

## 新功能
- Markdown 文档内本地路径自动转换为可点击链接，支持快速打开文件
- 中文用户默认启用 git.gh-proxy.com 镜像加速，提升 GitHub 访问体验

## 问题修复
- 修复更新安装后应用退出路径被阻塞导致无法完成安装的问题
- 修复插件面板始终为空的问题
- 修复文档路径链接化功能中文件目录前缀正则匹配失效的问题
- 修复设置面板首次打开不显示 Provider 列表的问题
- 修复 Markdown 文档查看器的路由跳转和标签页切换问题
- 修复 CodeX CLI Provider 配置错误
- 修复技能社区市场在 asar 打包模式下无响应的问题
- 修复技能社区搜索参数不一致的问题
- 限制文档路径只能在允许的目录内打开，防止越权访问
- 加固符号链接边界检查，防止文档和技能目录穿越
- 加固 Agent 配置和技能管理安全性
- 修复用户消息气泡高亮不支持主题切换的问题
- 优化应用图标清晰度和任务栏可见性
- 修复推荐插件标签去重问题，统一搜索机制

## 功能改进
- 重构 CodeX 认证方式，采用 experimental_bearer_token 内嵌 TOML，简化配置流程
- 重构技能社区市场改为直连 API，提升稳定性和速度
- 优化主题和内联路径显示样式
- 优化 Markdown 文档查看器标签页主题适配
