MindCraft-Agent v1.2.0 更新日志

新功能
- 文档标签页持久化：重启应用后自动恢复上次打开的文件标签页，包括 Markdown、图片、PDF 等文件类型

问题修复
- 修复 keep-alive 组件暂停时侧边栏运行状态丢失的严重 BUG：新增 codehubHasRunning 绕过 keep-alive 的 computed+watch 暂停，确保运行中图标始终准确显示
- 修复侧边栏"项目"图标运行中/已完成两种状态混淆显示的问题

功能改进
- 侧边栏运行状态视觉增强：运行中时图标快速闪烁 + 右上角小圆点跳动，已完成时图标脉冲 + 静止圆点，状态区分更直观
- 提问弹窗（AskQuestionDialog）改为绑定到父面板而非全局覆盖，不再遮挡整个界面
- 统一"运行中"和各 Provider 加载中的国际化文案
