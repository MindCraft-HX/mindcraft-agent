MindCraft-Agent v1.2.10 更新日志

macOS 优化
- 优化 macOS 窗口管理：关闭按钮改为隐藏窗口而非退出，激活时恢复隐藏窗口而非重新创建
- 完善 macOS 公证（Notarization）流程，确保打包签名通过公证校验
- 按平台（macOS/Windows/iOS）精细化调整打包与自动更新配置，新增端口检测工具
- 修复 macOS 应用图标显示问题，新增高清图标资源（512px）和 iOS 图标文件

环境探测增强
- 抽取打包环境下的共享 PATH 探测能力，统一维护可执行文件路径探测逻辑（新增 shellPathHelper 模块）
- CodeX 环境探测改用完整环境变量探测方式，支持手动配置可执行文件路径
- Claude 环境探测改用完整环境变量探测方式，提升环境检测兼容性

版本标准化
- 完善 Claude 版本标准化检测逻辑，统一各平台版本获取方式
- CodeX 设置页提取 extractVersion 公共函数，消除版本检测代码冗余

文档查看器优化
- 优化标签页溢出控制：支持鼠标滚轮横向滚动标签栏，方便在打开大量文档时快速浏览
- 新增标签页右键菜单：支持关闭当前标签页、关闭其他标签页、关闭全部标签页
