MindCraft-Agent v1.3.0 更新日志

功能新增
- 工作台分屏模式：支持文档查看器与代码面板分栏显示，可在同一窗口内同时浏览文档和操作代码
- CodeHub 外部标签投影：支持将 CodeHub 标签页作为独立面板嵌入工作台，提升多任务操作效率

问题修复
- 修复 CodeX 对话完成后恢复历史记录时，多余的注释文本未正确折叠的问题
- 修复 CodeX 对话恢复时剧本尾部未与完成状态正确对齐的问题
- 修复 CodeX 会话恢复后输入框无法使用的问题
- 修复 CodeX CLI 进程重启后会话归属关系丢失的问题
- 修复渲染器重载后 CodeX 运行中任务未能正确恢复的问题
- 修复 CodeHub 切换配置文件后标签页未按当前配置恢复的问题
- 修复工作台路由状态未正确保持的问题
- 修复 Diff 颜色可读性问题，确保代码差异在不同主题下清晰可辨
