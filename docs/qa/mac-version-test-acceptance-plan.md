> 版本：v1.2.6

## 0. 测试环境要求

### 测试机器配置

| 项目 | 最低要求 | 推荐 |
|------|---------|------|
| 系统版本 | macOS 12 (Monterey) | macOS 14 (Sonoma) 或更高 |
| 芯片架构 | Intel x64 / Apple Silicon (M1/M2/M3) | **两种架构各测一台** |
| 内存 | 8GB | 16GB+ |
| 磁盘空间 | 2GB 可用 | 5GB+ |

### 测试账号准备

| 服务 | 说明 |
|------|------|
| ClaudeCode API Key | 有效 Anthropic API Key（测试 ClaudeCode Agent） |
| CodeX API Key | 有效 OpenAI API Key（测试 CodeX Agent） |

### 测试包获取

- [ ] 从 COS 下载 DMG：`https://download.mindcraft.com.cn/MindCraft-Agent/installer/mac/`
- [ ] 或从内网共享获取 DMG/ZIP 安装包

## 11. 已知风险点（重点关注）

| R9 | 文件关联 (.md) | Mac 构建未配置文件关联（Windows 有），文档打开体验 | 🟢 P2 |
